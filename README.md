# Calculator Project

This is a simple calculator project created by Ajay Yadav. It is built using HTML, CSS, and JavaScript to provide basic mathematical operations in a user-friendly interface.

## Project Overview

- **Project Name:** Calculator
- **Author:** Ajay Yadav

## Features

- Addition, subtraction, multiplication, and division operations.
- User-friendly interface with clickable buttons.
- Responsive design for both desktop and mobile devices.

## Technologies Used

- HTML
- CSS
- JavaScript
